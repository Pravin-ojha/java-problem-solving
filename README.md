## 📌 About

This repository contains multiple Java programs written to improve logical thinking and coding skills.

All programs are stored in a single directory for simplicity.

---

## 🛠️ How to Run

1. Clone the repository

```
git clone https://github.com/Pravin-ojha/java-problem-solving.git
```

2. Open the folder

```
cd java-problem-solving
```

3. Compile and run any file

```
javac FileName.java
java FileName
```
